H: {% now "H" %} - Hour, 24-hour format.
i: {% now "i" %} - Minutes.
I: {% now "I" %} - Daylight Savings Time, whether it's in effect or not.
j: {% now "j" %} - Day of the month without leading zeros.
l: {% now "l" %} - Day of the week, textual, long.
L: {% now "L" %} - Boolean for whether it's a leap year.
m: {% now "m" %} - Month, 2 digits with leading zeros.
M: {% now "M" %} - Month, textual, 3 letters.
n: {% now "n" %} - Month without leading zeros.
N: {% now "N" %} - Month abbreviation in Associated Press style. Proprietary extension.
o: {% now "o" %} - ISO-8601 week-numbering year, corresponding to the ISO-8601 week number (W)
O: {% now "O" %} - Difference to Greenwich time in hours.
P: {% now "P" %} - Time, in 12-hour hours, minutes and 'a.m.'/'p.m.', with minutes left off if they're zero and the special-case strings 'midnight' and 'noon' if appropriate. Proprietary extension.
r: {% now "r" %} - RFC 2822 formatted date.
s: {% now "s" %} - Seconds, 2 digits with leading zeros.

t: {% now "t" %} - Number of days in the given month.
T: {% now "T" %} - Time zone of this machine.
u: {% now "u" %} - Microseconds.
U: {% now "U" %} - Seconds since the Unix Epoch (January 1 1970 00:00:00 UTC).
w: {% now "w" %} - Day of the week, digits without leading zeros.
W: {% now "W" %} - ISO-8601 week number of year, with weeks starting on Monday.
y: {% now "y" %} - Year, 2 digits.
Y: {% now "Y" %} - Year, 4 digits.
z: {% now "z" %} - Day of the year.
Z: {% now "Z" %} - Time zone offset in seconds. The offset for timezones west of UTC is always negative, and for those east of UTC is always positive.
